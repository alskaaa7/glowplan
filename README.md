# GlowPlan — Умный бьюти-календарь

Веб-приложение на Python (Flask) для планирования ухода за кожей.

---

## Структура проекта

```
glowplan/
├── app.py              # Точка входа, создание приложения     → Разработчик 2 (Backend)
├── models.py           # Модели базы данных (SQLAlchemy)      → Разработчик 2 (Backend)
├── requirements.txt    # Зависимости Python                   → Разработчик 3 (DevOps)
│
├── routes/
│   ├── auth.py         # Авторизация (вход, регистрация)      → Разработчик 2 (Backend)
│   ├── main.py         # Дашборд, профиль, чек-лист           → Разработчик 2 (Backend)
│   ├── products.py     # Личная полка продуктов               → Разработчик 2 (Backend)
│   ├── routine.py      # Конструктор рутины                   → Разработчик 2 (Backend)
│   └── admin.py        # Панель admin, роли, схемы гуру       → Разработчик 2 (Backend)
│
├── templates/          # HTML-шаблоны Jinja2                  → Разработчик 1 (Frontend)
│   ├── base.html       # Базовый шаблон с навигацией
│   ├── auth/           # Вход, регистрация
│   ├── main/           # Дашборд, профиль
│   ├── products/       # Полка, каталог, добавление
│   ├── routine/        # Конструктор расписания
│   └── admin/          # Панель, схемы гуру
│
└── static/
    ├── css/style.css   # Дизайн-система, вёрстка              → Разработчик 1 (Frontend)
    └── js/main.js      # UI-логика                            → Разработчик 1 (Frontend)
```

---

## Запуск проекта

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Запустить приложение
python app.py

# 3. Открыть в браузере
http://localhost:5000
```

При первом запуске автоматически создаётся база данных `instance/glowplan.db`
и добавляются тестовые продукты.

---

## Тестовый аккаунт администратора

После регистрации обычного аккаунта можно повысить роль через Flask shell:

```bash
flask shell
>>> from app import db
>>> from models import User
>>> u = User.query.filter_by(email='your@email.com').first()
>>> u.role = 'admin'
>>> db.session.commit()
```

---

## Роли пользователей

| Роль  | Возможности                                                  |
| ----- | ------------------------------------------------------------ |
| user  | Добавляет продукты, настраивает рутину, видит дашборд        |
| guru  | + Создаёт и публикует схемы ухода, управляет базой продуктов |
| admin | + Управляет пользователями и ролями                          |
